A real-time restaurant ordering web app where customers select items from a menu, enter a table number, and place an order.  
Kitchen staff receive orders instantly and can update order status.

## Features
- Customer menu + cart
- Table number capture
- Place orders to backend API
- Kitchen dashboard receives orders in real-time (Socket.IO)
- Update order status (Preparing / Ready / Served)

## Tech Stack
- Node.js
- Express
- Socket.IO
- HTML / CSS / JavaScript
